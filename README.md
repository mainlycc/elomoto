# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1VsfsjMSMKBr5eUBjdPiKvCBh7xfPC_eZ

## Run Locally

**Prerequisites:** Node.js (zalecane: aktualne LTS)


1. Install dependencies:
   `npm install`
2. Run the app (otworzy się automatycznie w przeglądarce):
   `npm run dev`

Po starcie aplikacja jest dostępna pod: `http://localhost:3000/`
